# 📧 Spam Email Classifier

A machine learning project that detects and filters **spam emails** based on their text content, using **Naive Bayes** and **Support Vector Machine (SVM)** classifiers.

## Overview

This project trains and compares two classic text-classification algorithms to distinguish spam from legitimate ("ham") emails:

- **Multinomial Naive Bayes** — a fast, probabilistic baseline that works well with word-frequency features.
- **Linear SVM** — finds the optimal boundary between spam and ham in TF-IDF feature space, and is the better-performing model on this dataset.

On the held-out test set, the final model (SVM) achieves **~99% accuracy** and a **~0.98 F1-score**.

## Dataset

- **5,728 emails** (after removing duplicates), each labeled as spam (`1`) or ham (`0`)
- Roughly **76% ham / 24% spam** — a realistic, moderately imbalanced split
- Each row contains the email's `Subject:` line plus its body text

## Pipeline

1. **Load & explore** the dataset (class balance, email length distribution, common words)
2. **Clean the text**: lowercase, strip the `Subject:` tag, remove punctuation/numbers, remove stopwords and very short tokens (using scikit-learn's built-in stopword list — no extra downloads needed)
3. **Split** into train/test sets (80/20, stratified by class)
4. **Vectorize** the cleaned text with **TF-IDF** (unigrams + bigrams, top 5,000 features)
5. **Train** a Multinomial Naive Bayes model and a Linear SVM model
6. **Evaluate** both with accuracy, precision, recall, F1-score, confusion matrices, and 5-fold cross-validation
7. **Test** the best model on brand-new example emails
8. **Save** the trained model and vectorizer with `joblib` for reuse

## Using the Saved Model

```python
import joblib
import re
from sklearn.feature_extraction.text import ENGLISH_STOP_WORDS

model = joblib.load("spam_classifier_model.pkl")
vectorizer = joblib.load("tfidf_vectorizer.pkl")

def clean_text(text):
    text = text.lower()
    text = re.sub(r"^subject:", "", text)
    text = re.sub(r"[^a-z\s]", " ", text)
    text = re.sub(r"\s+", " ", text).strip()
    words = [w for w in text.split() if w not in ENGLISH_STOP_WORDS and len(w) > 2]
    return " ".join(words)

email = "Subject: You won a free prize! Click here to claim now!!!"
vec = vectorizer.transform([clean_text(email)])
prediction = model.predict(vec)[0]
print("SPAM" if prediction == 1 else "HAM")
```

## Possible Improvements

- Try additional algorithms: Logistic Regression, Random Forest, or a fine-tuned transformer (e.g. BERT)
- Use character n-grams to catch obfuscated spam tricks (e.g. "v1agra", "fr33")
- Add metadata features: sender domain, number of links, presence of attachments
- Apply class-weighting or resampling (e.g. SMOTE) if used on a more imbalanced dataset
- Deploy as a simple API or browser extension for real-time filtering
